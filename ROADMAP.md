# Protocol Roadmap

This document outlines the planned evolution of the protocol.

## Version 1.x Goals

- stabilize terminology
- refine wire format definitions
- finalize reason code registry
- define conformance test cases

## Future Work

Possible future extensions may include:

- multi-registry trust federation
- distributed governance evidence validation
- registry discovery mechanisms
- protocol negotiation for future versions

## Stability Policy

Minor versions may introduce non-breaking extensions.

Major versions may introduce architectural or protocol-level changes.
